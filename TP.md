# TP n°4


Créer un nouveau pipe nommé NamePipe permettant de filtrer les jeux en fonction 
de leur nom. 

Par exemple, si je tape dans ma barre d'input la lettre 'm' alors tous les jeux dont 
le nom contient la lettre m, apparaissent, les autres sont masqués. 

Autre exemple, si je tape dans ma barre d'input les lettres 'ma', alors seuls les jeux 
contenant cette séquence de lettres sont affichés, les autres sont masqués. 


ATTENTION : Le filtre ne doit pas empêcher le filtre sur les plateformes de fonctionner. 
BONUS: Faire en sorte que la recherche soit insensible à la casse.
